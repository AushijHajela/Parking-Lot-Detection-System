# Parking-Lot-Detection-System

A real-time smart parking solution using Computer Vision

📌 Overview

Finding a parking spot in crowded areas is frustrating and time-consuming. This project aims to solve that problem by automatically detecting available parking spaces using computer vision.

The system analyzes live video or images and identifies whether parking slots are occupied or vacant, helping users and authorities manage parking efficiently.

🎯 Features

🔍 Real-time parking slot detection

📊 Occupied vs Vacant space classification

🎥 Works with video streams or recorded footage

🧠 Machine Learning / Computer Vision based approach

📈 Visual output with highlighted parking spots

⚡ Fast and efficient processing

🛠️ Tech Stack

Python
OpenCV – Image processing
NumPy – Data handling
Machine Learning / Deep Learning (optional)
Streamlit (optional) – UI for real-time interaction

⚙️ How It Works

Input video/image of a parking area

Predefined parking slot regions are mapped

Each slot is analyzed using image processing techniques

System determines if a slot is empty or occupied

Output is displayed with:

🟢 Green → Available
🔴 Red → Occupied
